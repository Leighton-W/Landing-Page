# Landing Page

A static landing page built using HTML and CSS.
This project was created as part of my self-directed web development learning and focuses on page structure, CSS layout, Flexbox and visual styling.

## Features

* Header with navigation links
* Hero section with call-to-action button
* Information section with multiple content cards
* Quote/testimonial section
* Call-to-action banner
* Footer section
* Structured page layout using Flexbox
* Consistent spacing, colours and typography

## Technologies Used

* HTML
* CSS

## What I Learned

Through this project I developed my understanding of how to structure a complete webpage using semantic HTML elements and reusable CSS classes.
I gained experience using Flexbox to align content, create sections, manage spacing and build a clean multi-section page layout. The project also helped me practise applying consistent colours, typography and visual hierarchy across a webpage.
